**Hook format (from research):**
- Event types: PreToolUse, PostToolUse, Stop, Notification, etc.
- Matchers: regex on tool name ("Bash", "Edit|Write")
- Hook types: command (shell script), http (webhook URL)
- Command hooks: { type: "command", command: "/path/to/script.sh", timeout: N }
- Exit code 0 = success, 2 = blocking error
- "if" field for permission-rule filter on PreToolUse
</context>
