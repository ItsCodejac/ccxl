<task type="auto">
  <name>Task 1: Create cross-tool adapter system and all three adapters</name>
  <files>src/generators/cross-tool/adapter.ts, src/generators/cross-tool/cursor.ts, src/generators/cross-tool/copilot.ts, src/generators/cross-tool/windsurf.ts, src/generators/cross-tool/shared-rules.ts, src/generators/cross-tool/index.ts</files>
  <action>
  1. Create src/generators/cross-tool/shared-rules.ts:
     Define shared rule content that gets adapted per tool.
     Rules derived from ProjectAnalysis:
     - "general" (always on): project overview — languages, frameworks, package manager
     - "code-style" (always on): coding standards derived from detected linters/formatters
     - "testing" (glob: test files): test framework conventions
     - "api" (glob: route files, conditional on backend): API patterns
     - "components" (glob: component files, conditional on frontend): UI component patterns

     Each rule is: { id, title, description, globs?, alwaysOn, condition?, body(analysis) }

  2. Create src/generators/cross-tool/adapter.ts:
     Define CrossToolAdapter interface: { toolName, generateLegacy(analysis): GeneratedFile, generateModern(analysis): GeneratedFile[] }

  3. Create src/generators/cross-tool/cursor.ts:
     Implements CrossToolAdapter.
     - Legacy: .cursorrules — concatenate all rule bodies as plain markdown
     - Modern: .cursor/rules/{id}.mdc files with frontmatter: description, globs, alwaysApply

  4. Create src/generators/cross-tool/copilot.ts:
     - Legacy: .github/copilot-instructions.md — all rules concatenated
     - Modern: .github/instructions/{id}.instructions.md with frontmatter: applyTo

  5. Create src/generators/cross-tool/windsurf.ts:
     - Legacy: .windsurfrules — all rules concatenated
     - Modern: .windsurf/rules/{id}.md with frontmatter: trigger, globs, description

  6. Create src/generators/cross-tool/index.ts:
     Export crossToolGenerator implementing Generator interface.
     Reads --target option from analysis/options to decide which tools to generate for.
     Default: generate for all tools.
     Produces both legacy and modern formats for each tool.
  </action>
  <verify>npx tsc --noEmit passes</verify>
  <done>Three adapters generate both legacy and modern rule formats from shared rule definitions</done>
</task>
