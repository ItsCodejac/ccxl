  <action>
  1. Create src/registry/search.ts:
     - async function searchPackages(query: string): Promise<SearchResult[]>
     - Searches GitHub via API: GET https://api.github.com/search/repositories?q={query}+topic:ccxl-package
     - Returns: { name, fullName, description, stars, url }[]
     - Uses GitHub topic "ccxl-package" as the convention for discoverable packages
     - Falls back to name-based search if no topic results

  2. Create src/registry/list.ts:
     - async function listInstalled(root: string): Promise<InstalledPackage[]>
     - Reads .claude/ccxl-packages.json
     - Returns installed packages with metadata
     - Returns empty array if file doesn't exist

  3. Update src/commands/registry.ts:
     - `ccxl registry search <query>` — calls searchPackages, prints results as table (name, description, stars)
     - `ccxl registry list` — calls listInstalled, prints installed packages (name, version, files count)
     - `ccxl registry browse` — opens Ink TUI with search input + results list (use SelectPrompt)
       When user selects a package, show description and "Install?" confirm prompt
       On confirm, run installPackage
     - `ccxl registry publish` — print instructions for now:
       "To publish a ccxl package:
       1. Create ccxl-package.json in your repo root
       2. Add topic 'ccxl-package' to your GitHub repo
       3. Push to GitHub — it's immediately installable via `ccxl install user/repo`"
