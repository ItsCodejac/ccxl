<task type="auto">
  <name>Task 1: Create base config system with inheritance</name>
  <files>src/governance/base-config.ts, src/governance/types.ts, src/governance/index.ts, src/commands/config.ts</files>
  <action>
  1. Create src/governance/types.ts:
     - BaseConfig: { name: string, source: string, version: string, policies: PolicyRule[] }
     - PolicyRule: { type: 'require'|'deny'|'enforce', target: string, value: string, message?: string }
       Examples:
       - { type: 'require', target: 'permission:deny', value: 'Read(.env*)' } — must have this deny rule
       - { type: 'deny', target: 'permission:allow', value: 'Bash(rm -rf *)' } — must not allow this
       - { type: 'enforce', target: 'hook:PreToolUse', value: 'block-dangerous-git' } — must have this hook
     - GovernanceReport: { compliant: boolean, violations: PolicyViolation[], policies: number }

  2. Create src/governance/base-config.ts:
     - async function loadBaseConfig(root: string): Promise<BaseConfig | null>
       Reads .claude/base-config.json if it exists
     - async function checkCompliance(root: string, baseConfig: BaseConfig): Promise<GovernanceReport>
       Reads current settings.json and checks each policy rule
       Returns compliant/violations
     - async function applyBaseConfig(root: string, baseConfig: BaseConfig): Promise<void>
       Merges base config policies into local settings using merge logic

  3. Create src/governance/index.ts with exports.

  4. Update src/commands/config.ts:
     - `ccxl config show` — shows current ccxl config (base config if set, policies)
     - `ccxl config set base <user/repo>` — installs a base config package as the project's base
       Fetches the repo, reads its base-config.json, saves to .claude/base-config.json
     - `ccxl config check` — runs compliance check against base config
     - `ccxl config reset` — removes base config
  </action>
  <verify>npx tsc --noEmit passes, npm run build succeeds</verify>
  <done>Base config system with policy rules, compliance checking, and config commands</done>
</task>
